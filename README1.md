# Fitness Tracker System

This is a Java-based Fitness Tracker System developed as an academic project.

## Features
- User registration and profile management
- Activity tracking
- Basic fitness data handling
- Console-based interaction

## Technologies Used
- Language: Java
- IDE: Apache Netbeans
- Type: Academic Project

## Project Structure
- Multiple Java classes handling different system components
- Modular and object-oriented design

## Purpose
This project was built to practice Java programming, OOP concepts, and system design fundamentals.
